# Tesla Parts UA - Next.js Shop

This is a Next.js version of the Tesla Parts UA shop, migrated from a Vite/React-Router application.

## Getting Started

1. Navigate to the `new-tesla-shop` directory:
   ```bash
   cd new-tesla-shop
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

## Original Application

The original Vite/React application is located in `new-tesla-frontend`.
